# Student Management System

## Giới thiệu

Student Management System là hệ thống quản lý sinh viên được xây dựng nhằm hỗ trợ nhà trường trong việc quản lý thông tin sinh viên, lớp học, môn học và kết quả học tập.

Hệ thống cung cấp giao diện trực quan giúp quản trị viên dễ dàng thực hiện các nghiệp vụ quản lý, tìm kiếm và thống kê dữ liệu.

## Chức năng chính

### Quản lý sinh viên

* Thêm sinh viên mới
* Cập nhật thông tin sinh viên
* Xóa sinh viên
* Tìm kiếm sinh viên
* Xem danh sách sinh viên

### Quản lý lớp học

* Thêm, sửa, xóa lớp học
* Gán sinh viên vào lớp
* Theo dõi danh sách sinh viên theo lớp

### Quản lý môn học

* Thêm, sửa, xóa môn học
* Quản lý thông tin môn học

### Quản lý điểm

* Nhập điểm sinh viên
* Cập nhật điểm
* Theo dõi kết quả học tập
* Thống kê học lực

### Thống kê và báo cáo

* Thống kê số lượng sinh viên
* Thống kê theo lớp
* Báo cáo kết quả học tập

## Hướng dẫn cài đặt

### Clone project

```bash
git clone https://github.com/thanhdao03/IT_FLM.git
```

### Frontend

```bash
cd frontend
npm install
npm start
```

### Backend

```bash
cd backend
mvn spring-boot:run
```

### Database

* Tạo database MySQL.
* Import file SQL trong thư mục database.
* Cập nhật cấu hình trong application.properties.
